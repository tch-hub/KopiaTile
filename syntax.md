# KopiaTile 構文ルール

KopiaTileは、各ピクセルの色を計算して描画する**ピクセル・シェーダー**のような仕組みを持つ、Lua言語ベースの実行環境です。

## 基本的な仕組み

ユーザーが記述したコードは、内部で暗黙的に `_evaluate_pixel(x, y, width, height)` という関数の中に組み込まれて実行されます。この関数は、描画領域のすべてのピクセルに対して1回ずつ呼び出されます。

コードの最終的な目的は、**そのピクセルに塗るべき色の番号(カラーコード)を `return` する** ことです。

```lua
-- 例: すべてのピクセルを白(1)で塗る
return 1
```

## 座標系（グリッド）

KopiaTileは標準で **9x9** のグリッドを想定した座標系を使用しています。

- **X座標 (`x`)**: 左端が `-4`、右端が `4` （中心が `0`）
- **Y座標 (`y`)**: 下端が `-4`、上端が `4` （中心が `0`）

## 利用可能な変数

コード内では、現在評価されているピクセルの状態を示す以下の変数を利用できます。

| 変数名   | 型       | 説明                                        |
| :------- | :------- | :------------------------------------------ |
| `x`      | `number` | 現在のピクセルの X座標 (-4 から 4 まで加算) |
| `y`      | `number` | 現在のピクセルの Y座標 (4 から -4 まで減算) |
| `width`  | `number` | グリッドの全幅                              |
| `height` | `number` | グリッドの全高                              |

## 定数（色）

よく使う色は、数値を直接指定する代わりに以下の定数文字列を使って指定できます。

| 定数名        | 値  | 色   |
| :------------ | :-- | :--- |
| `TRANSPARENT` | `0` | 透明 |
| `WHITE`       | `1` | 白   |
| `BLACK`       | `2` | 黒   |
| `RED`         | `3` | 赤   |
| `BLUE`        | `4` | 青   |

## 組み込み関数

よく使う数学関数は、直接呼び出すことができるように用意されています。

| 関数名 | 引数  | 説明                                        |
| :----- | :---- | :------------------------------------------ |
| `abs`  | `(n)` | `n` の絶対値を返します（`math.abs` と同等） |

## 戻り値（色）について

- ピクセルの色を決定するためには、数値（`number`）または色の定数（`WHITE`など）を `return` してください。
- `return` を省略した場合や、条件分岐で何も返さなかった（`nil` が返った）場合、デフォルトの色として `WHITE`（`1`）が採用されます。

## コード例

**例1: 中心を通る十字を描く**

```lua
if x == 0 or y == 0 then
    return BLACK -- 十字は黒
else
    return WHITE -- それ以外は白
end
```

**例2: X座標とY座標の値を使った市松模様**

```lua
if (x + y) % 2 == 0 then
    return BLACK
else
    return WHITE
end
```

**例3: 座標を使った単純なグラデーション風**

```lua
-- 返り値に依存しますが、座標をそのまま色番号として使うことも可能です
return (abs(x) + abs(y)) % 5 -- 0から4までの値になるように調整
```

**例4: `for` ループを使ったグリッド（網目模様）**

```lua
-- -4 から 4 まで、2 刻みでループします
for i = -4, 4, 2 do
    if x == i or y == i then
        return BLUE -- 等間隔で青色の直線を引く
    end
end

return TRANSPARENT
```

**例5: 円を描く（中心からの距離による条件分岐）**

```lua
-- 中心 (0, 0) からの距離の2乗を計算します
local dist_sq = x * x + y * y

-- 半径が2以下の円の中（距離の2乗が4以下）を赤く塗る
if dist_sq <= 4 then
    return RED
else
    return TRANSPARENT
end
```

**例6: 円形のグラデーションパターン**

```lua
-- 中心からの距離に応じて色を順番に切り替えるグラデーション
local dist_sq = x * x + y * y

if dist_sq <= 1 then
    return WHITE  -- 中心付近
elseif dist_sq <= 4 then
    return RED    -- ひとつ外側
elseif dist_sq <= 9 then
    return BLUE   -- さらに外側
else
    return BLACK  -- 一番外側
end
```

**例7: `for` ループを使った波紋（同心円）**

```lua
local dist_sq = x * x + y * y

-- 半径1〜4の円周を描く
for r = 1, 4 do
    local r_sq = r * r
    -- 距離の2乗が半径の2乗と完全にピタリと一致するピクセルを抽出する
    if dist_sq == r_sq then
        return BLUE
    end
end

return TRANSPARENT
```

## 注意点

- 言語は **Lua** として評価されます。関数定義やテーブル、標準ライブラリ（`math`など）が一部利用可能です。
- スクリプトはWeb Worker（バックグラウンド）上で非同期実行されるため、無限ループなどが生じてもブラウザはフリーズしません。ただし、一定時間（約2秒）実行が完了しない場合は強制終了されエラーとなります。
